## Documentation has code

Documentation writers sometimes like to include blocks of nonsense ("code"). Here's some of that, just using standard markdown code fences. Be sure to include any language grammars for syntax highlight that aren't already available in the `GrammarProvider` in `layouts/index.jsx`.

```go
// ...

caCert, _ := ioutil.ReadFile("ca.crt")
caCertPool := x509.NewCertPool()
caCertPool.AppendCertsFromPEM(caCert)

tlsConfig := &tls.Config{
    ClientCAs: caCertPool,
    ClientAuth: tls.RequireAndVerifyClientCert,
}
tlsConfig.BuildNameToCertificate()

server := &http.Server{
    Addr:      ":9443",
    TLSConfig: tlsConfig,
}

server.ListenAndServeTLS("server.crt", "server.key")

// ...
```

If you want to disable the automatic copy behavior, add the `nocopy` attribute to your code fence.

```python nocopy
def dont_copy_me():
    return "thanks"
```

If you need custom copy text (eg. to exclude command prompts or join commands), import and use the `docs/CodeBlock` component directly.
